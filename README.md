# PetLorem

Projeto de página institucional fictícia para pet shop, pensado para treinar fundamentos de desenvolvimento web com uma estrutura simples, visual e acessível para quem está começando.

## Por que este projeto está na stack recomendada para iniciantes

O PetLorem foi escolhido para alunos iniciantes porque permite praticar, em um único projeto, os principais blocos de uma página real:

- Estrutura semântica com HTML
- Estilização básica com CSS
- Uso de componentes prontos com Bootstrap
- Organização de seções comuns de sites institucionais
- Boas práticas de leitura e manutenção de código

É um projeto com resultado visual rápido, o que aumenta motivação e ajuda no aprendizado progressivo.

## Propósito do projeto

- Simular o site de uma empresa (pet shop) com foco em apresentação de serviços
- Reproduzir de forma fiel a interface já publicada, usando componentes do Bootstrap 5
- Trabalhar layout e hierarquia de conteúdo
- Desenvolver noções de responsividade
- Exercitar reutilização de classes e componentes
- Preparar base para próximos passos com JavaScript e integração com back-end

## O que o aluno aprende aqui

- Montar um layout com navbar, carrossel, cards e formulário
- Aplicar classes utilitárias do Bootstrap para espaçamento e alinhamento
- Entender como traduzir um layout pronto para uma implementação equivalente com Bootstrap 5
- Customizar aparência com CSS próprio
- Organizar assets em pasta de imagens
- Entender a separação de responsabilidades entre HTML e CSS

## Escopo de reprodução com Bootstrap 5

Para manter a proposta do projeto, os alunos devem reconstruir a página publicada preservando estrutura e experiência visual.

Componentes e blocos esperados:

- Navbar responsiva com colapso no mobile
- Carousel com indicadores e controles
- Seção institucional (Sobre Nós)
- Grade de cards de serviços
- Seção de equipe com imagens em formato circular
- Formulário de contato estilizado
- Rodapé com informações e links sociais

Critério principal: manter equivalência visual e estrutural com a versão publicada, sem depender de copiar código pronto linha por linha.

## Como executar

1. Clone o repositório.
2. Abra a pasta do projeto no VS Code.
3. Abra o arquivo index.html no navegador.

Opcionalmente, use a extensão Live Server para recarregamento automático durante as alterações.

## Sugestão de trilha para aula

1. Navegar na versão publicada e mapear as seções visíveis
2. Definir quais componentes do Bootstrap 5 serão usados em cada seção
3. Reproduzir a estrutura HTML por blocos (navbar, carousel, cards, formulário)
4. Ajustar estilos complementares no arquivo style.css
5. Refinar responsividade e consistência visual

## Desafios recomendados para os alunos

- Trocar textos de placeholder por conteúdo real
- Melhorar acessibilidade com atributos `alt` descritivos e labels mais claras
- Adicionar validação básica de formulário com JavaScript
- Criar seção de depoimentos de clientes
- Implementar botão de CTA com destaque visual

## Critérios de avaliação sugeridos

- Clareza e organização do HTML
- Consistência visual e uso do CSS
- Qualidade da adaptação para telas menores
- Legibilidade do código
- Capacidade de evoluir o projeto sem quebrar a estrutura existente

## Próximo passo natural

Depois deste projeto, os alunos podem avançar para:

- Interatividade com JavaScript (menu dinâmico, validação e feedback visual)
- Consumo de dados em JSON para renderizar cards de serviços
- Publicação em plataformas como GitHub Pages
